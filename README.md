# markdown-to-html

A utility that converts Markdown files to HTML

## Getting Started

### Prerequisites

- Python 3.10+

### Installation

```bash
git clone https://github.com/YOUR_USERNAME/markdown-to-html.git
cd markdown-to-html
```

### Run

```bash
python main.py
```
